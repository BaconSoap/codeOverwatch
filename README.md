##install

```bash
npm install -g forever bower
cd client && npm install && cd ../server && bower install && npm install
```

then go config it.

##config

move config.json to secrets/config.json and populate it with your info

##edit/run

in client, do

```bash
grunt w
```

in server, do

```bash
./runServer
```
